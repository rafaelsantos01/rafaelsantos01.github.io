FKSYSTEM


Aplicação desenvolvida para o gerenciamento de O.S.

Objetivo do software é gerenciar um negócio que trabalha com O.S.

O mesmo vai realizar o controle e cadastro de estoque, contas a pagar,

contas a receber, funcionários, fornecedores, e claro controle de O.S.

O sistem possui um banco de dados MySQL e foi projetado em Java
